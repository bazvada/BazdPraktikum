# Описание.

## django_sprint4.

## Используемые технологии:
* Python 3.7
* SQlite3
* Django 3.2

## Что делает проект

Сервис предназначен для размещения пользовательских публикаций в формате личного блога.
Пользователи могут создавать и хранить записи, а сам проект демонстрирует работу Django-приложения с базовой социальной функциональностью.

## Инструкция по запуску:

* Скопируйте проект из репозитория и откройте директорию с файлами:

       Git clone git@github.com:bazvada/BazdPraktikum.git
        cd BazdPraktikum.git

* Создайте отдельное виртуальное окружение для зависимостей проекта:

        python -m venv venv
        source venv/Scripts/activate

* Установите все необходимые пакеты:

        python -m pip install --upgrade pip
        pip install -r requirements.txt

* Подготовьте базу данных, применив миграции:

        python manage.py migrate


* После этого можно запускать встроенный сервер Django:

        python manage.py runserver

* По умолчанию приложение откроется по адресу:

        http://127.0.0.1:8000/
